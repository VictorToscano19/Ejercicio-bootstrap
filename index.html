Boosttrap-Intro



Clases principales y de mayor uso en Bootstrap

A continuacion te presento algunas clases importantes para utilizar en bootstrap

class="container"

Ocupa las 12 columnas del grid que proporciona CSS, pero deja un margen a los costados

class="container-fluid"

Ocupa el tamaño completo del viewport

Class="Row"

Indica que el elemento contenedor se comporta como una fila dentro del grid

Class="col-n/xx"

Dos permiten establecer el numero de columnas que queremos que ocupe un elemento/componente.
Se pueden delimintar tanto con numero como con nomenclaturas predefinidas
Ejemplos: sm, md, lg, xl, xxl.
